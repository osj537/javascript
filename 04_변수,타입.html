변수선언하기

js는 동적타입을 사용하기 때문에 변수를 선언할때 자료형을 선언하지 않음
var 예약어를 이용해서 변수를 선언
선언방식 : var 변수명;

변수의 타입은 변수에 대입된 값을 기준으로 설정
한개 변수에 다수의 자료형 값을 지정할 수 있다.

    타입별 리터럴 표시
  • string : 문자열, "" 또는 '' 으로 표시, 예시) "화이팅" = '화이팅'
  • number : 숫자(정수, 실수), 180, 65.4 표시, 실수/정수 구분없이 저장
  • boolean : true/false
  • object : new Object명(), {ket:value, key2:value2,...} new Data(), new RegExp(),....
  • Array : new Array(), [값,값2,값3,...]
  • function : function ([배개변수,...]){[return 값;]},([매개변수,...])=>{[return값;]}
  • null : 값 없음을 의도적으로 표시할때 사용 ,null
  • undefined : 자동대입, 선언된 변수를 초기화 하지 않으면 저장되는 타입

var로 선언된 변수의 scope

전역변수 : function외부에 선언된 모든 변수 -> window의 맴버변수로 등록됨
지역변수 : function내부에 선언된 변수 -> function에서만 사용
js는 블록범위를 사용하지 않음

변수의 중복선언을 허용함

변수 호이스팅

변수가 선언된 라인을 읽기전에 변수를 호출할 수 있게 해주는 것!

ES6에서 let,const예약어 추가

1.중복선언을 허용하지 않음
2.블록범위를 준수함
3.호이스팅 허용하지 않음 -> 읽는 것을 막아버림

선언방법 let 변수명; -> 변경이 가능한 변수(일반변수)
const 변수명; -> 변경이 불가능한 변수(상수)
