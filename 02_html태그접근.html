html태그에 접근하기

javascript에서 태그를 한개 객체로 취급함. -> DOM
js가 제공하는 document객체를 이용해서 html에 있는 태그를 객체로 가져올 수 있음

  • getElementById() : 태그에 설정된 id속성을 기준으로 태그를 가져옴 -> 한개만
  • getElementsByClassName(): 태그에 설정된 class속성값을 기준으로 태그를 가져옴 -> 다수 태그 (0개 이상)
  • getElementsByName() : 태그이름으로 태그를 가져옴 -> 다수 (0개 이상)
  • getElementsByTagName() : 태그에 설정된 name속성값(input)을 기준으로 태그를 가져옴 -> 다수 (0개 이상)
  • querySelector() : 인수에 설정된 선택자에 해당하는 태그 한개를 가져옴 -> 0~1개
  • querySelectorAll() : 인수에 설정된 선택자에 해당하는 태그 한개이상을 가져옴 -> 다수 (0개 이상)
  • 다수 : 리스트형태 / 한개 HtmlElement
