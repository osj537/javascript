데이터 출력하기

html페이지나 개발자도구에 원하는 데이터 출력하기

  • html페이지에 출력 -> 동적으로 문구를 수정할때
    write(), HTMLElment.innerText||inneerHTML속성을 이용
  • console창에 출력 -> 경고, 발생한 에러 정보를 출력하거나 디버깅용 데이터를 출력할때
    console.log(), console.error(), console.warn(), console.dir(), console.table()
  • 알림창으로 출력 -> 알림 메세지를 출력할때
    alert() 메소드 이용

console창에 출력하기

js가 제공하는 console객체를 이용,
개발자도구 console탭에서 확인이 가능
js코드 관련된 사항을 출력

알람창으로 데이터 출력하기

alert() 함수를 호출 -> window객체가 제공
window객체를 browser마다 생성
window객체는 최상위 객체로 메소드등을 사용할때 생략하고 사용할 수 있다.
window.alert("출력문구");
alert("출력문구");

html페이지에 문구추가하기

document.write()함수

특정태그에 문구를 작성, 수정

HTMLElement객체의 innerText, inneerHTML속성을 이용
innerText : 모든 대입된 문자를 문자열로 처리
innerHTML : 태그형식의 문자는 태그로 해석, 일반문자는 일반문자로 해석
