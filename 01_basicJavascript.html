자바스크립트 적용하기

  • inline방식 : 이벤트 속성에 자바스크립트 구문을 작성하는 것
  • internal방식 : html내 script태그에 자바스크립트 구문을 작성하는 것 script태그를 다수 작성할 수 있고, 선언위치는 제한이없다.
  • external방식 : *.js파일을 생성하고 파일내부에 자바스크립트 구문을 작성하는 것 script태그를 이용ㅌㄴ해서 파일을 불러와 사용

자바스크립트는 인터프리터 방식으로 위에서 순차적으로 발생함
코드의 위치가 중요하다.
